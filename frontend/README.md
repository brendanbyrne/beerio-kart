# Frontend

React web app for Beerio Kart, built with [Vite](https://vite.dev/).

## Tech

- **Framework:** React + TypeScript
- **Bundler:** Vite
- **Package manager:** Bun

## Running

```sh
bun install
bun dev
```

Dev server starts on `http://localhost:5173`. API requests to `/api` are proxied to the backend at `http://localhost:3000`.
